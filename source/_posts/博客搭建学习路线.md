## 你需要学什么？

按优先级从高到低排列：

### 第一优先级：Markdown（30 分钟）

这是写文章的基本功，超简单。

- 推荐教程：[Markdown 官方教程](https://markdown.com.cn/) — 10 分钟看完
- 核心记住：`#` 标题、`**加粗**`、`*斜体*`、`-` 列表、` ```代码块``` `、`![图片](url)`
- 边写边练即可，不需要背

### 第二优先级：YAML 语法（15 分钟）

博客的配置文件（`_config.yml`、`_config.butterfly.yml`）和文章的 Front-matter 都用的 YAML。

- 推荐：[YAML 入门教程](https://www.ruanyifeng.com/blog/2016/07/yaml.html) — 阮一峰写的中文教程
- 核心记住：`key: value`、`-` 列表、缩进用空格
- 15 分钟足够

### 第三优先级：Hexo 基础（1 小时）

- 官网文档：[Hexo 中文文档](https://hexo.io/zh-cn/docs/)
- 重点看这 4 页：
  - 建站（Setup）
  - 写作（Writing）— Front-matter 格式
  - 指令（Commands）— `hexo new / clean / generate / server / deploy`
  - 配置（Configuration）— `_config.yml` 各字段含义

### 第四优先级：Butterfly 主题（按需查阅）

- 官方文档：[Butterfly 文档](https://butterfly.js.org/)（中文）
- 不需要通读，用到什么查什么：
  - 改导航菜单 → 看 Menu 章节
  - 加评论 → 看 Comments 章节
  - 改样式 → 看 CSS 自定义章节
  - 开启搜索 → 看 Search 章节

### 第五优先级：Git / GitHub（可选，部署时需要）

如果要部署到 GitHub Pages：

- [Git 教程 — 廖雪峰](https://www.liaoxuefeng.com/wiki/896043488029600) — 只看前 4 章
- 核心命令：`git clone`、`git add`、`git commit`、`git push`

---

## 推荐学习顺序

```
Markdown  ⏱ 30分钟
    ↓
YAML      ⏱ 15分钟
    ↓
Hexo      ⏱ 1小时
    ↓
Butterfly ⏱ 用到再看
    ↓
Git       ⏱ 2小时（部署前学）
```

---

## 总结

| 技能 | 必学 | 耗时 | 用途 |
|------|:---:|------|------|
| Markdown | ✅ | 30 min | 写文章 |
| YAML | ✅ | 15 min | 改配置 |
| Hexo 命令 | ✅ | 1 h | 建站部署 |
| Butterfly 配置 | ✅ | 用到再看 | 主题定制 |
| Git | ⏳ | 2 h | 部署上线 |
| CSS/HTML | ❌ | - | 深度自定义 |
| JavaScript | ❌ | - | 插件开发 |

**核心就三个：Markdown 写文章、YAML 改配置、那几条 hexo 命令。** 其他的用到再学。

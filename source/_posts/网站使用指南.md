## 快速开始

### 创建新文章

在项目目录 `D:\my-blog` 下执行：

```bash
hexo new "文章标题"
```

文章会生成在 `source/_posts/` 目录下。

### 文章格式

每篇文章以 **Front-matter** 开头，用于声明元信息：

```markdown
---
title: 文章标题
date: 2026-05-19
tags: [标签1, 标签2]
categories: 分类名
---
正文内容...
```

> **注意**：`tags` 和 `categories` 区分大小写。同一分类下的文章会自动归类。

### 重新生成网站

修改文章后需重新生成才能看到效果：

```bash
hexo clean && hexo generate
```

然后刷新浏览器 `http://localhost:4000/`。

## 常用命令

| 命令 | 作用 |
|------|------|
| `hexo new "标题"` | 新建文章 |
| `hexo clean` | 清除缓存和生成文件 |
| `hexo generate` | 生成静态文件 |
| `hexo server` | 启动本地预览服务器 |
| `hexo deploy` | 部署到远程服务器 |

简写形式：

```bash
hexo n "标题"    # new
hexo cl          # clean
hexo g           # generate
hexo s           # server
hexo d           # deploy
```


## Butterfly 主题配置

主题配置文件位于 `D:\my-blog\_config.butterfly.yml`。

### 修改网站信息

编辑 `D:\my-blog\_config.yml`：

```yaml
title: 你的网站名
subtitle: '副标题'
description: '网站描述'
author: 你的名字
language: zh-CN
```

### 开启导航菜单

在 `_config.butterfly.yml` 中配置 menu：

```yaml
menu:
  首页: / || fas fa-home
  归档: /archives/ || fas fa-archive
  标签: /tags/ || fas fa-tags
  分类: /categories/ || fas fa-folder
  # 关于: /about/ || fas fa-heart
```

### 暗色模式

Butterfly 默认支持暗色模式，点击页面右下角按钮即可切换。

### 社交链接

在 `_config.butterfly.yml` 中配置：

```yaml
social:
  fab fa-github: https://github.com/你的用户名 || GitHub
  fab fa-twitter: https://twitter.com/你的用户名 || Twitter
```

### 开启搜索

在 `_config.butterfly.yml` 中：

```yaml
local_search:
  enable: true
```

### 开启评论

Butterfly 支持多种评论系统。以 Twikoo 为例：

```yaml
comments:
  use: Twikoo
twikoo:
  envId: 你的环境ID
  region: ap-shanghai
```

## 新建独立页面

### 标签页

```bash
hexo new page tags
```

编辑 `source/tags/index.md`，添加：

```markdown
---
title: 标签
type: tags
---
```

### 分类页

```bash
hexo new page categories
```

编辑 `source/categories/index.md`，添加：

```markdown
---
title: 分类
type: categories
---
```

### 关于页

```bash
hexo new page about
```

编辑 `source/about/index.md` 自由书写。

## 部署到线上

### GitHub Pages（免费）

1. 安装部署插件：

```bash
cnpm install hexo-deployer-git --save
```

2. 修改 `_config.yml`：

```yaml
deploy:
  type: git
  repo: https://github.com/用户名/用户名.github.io.git
  branch: main
```

3. 执行部署：

```bash
hexo clean && hexo deploy
```
